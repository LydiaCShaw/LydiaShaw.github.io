Traffic Information Management System

Introduction

According the flow chart below, the TIM system should construct drivers database so that one can upload license information and violation record.Once the violation record is uploaded to the database, the information can be searched out.

Meanwhile,when a vehicle with this license plate passes the camera, plate number will be recognized. If results were obtained successfully, TIM system will match the historical record in the database. As long as there is a violation record, the program will pop up a prompt and display the owner's information.

To develop this program,Visual Studio 2013, MySQL and OpenCV libraries were used in TIM system. We used Visual Studio to create a VB Windows Form program and connect to the MySQL database.After encapsulating,we made a simple executable program.One can upload the information and realize the search function directly in executable program.

The OpenCV libraries can realize the recognition of license plate images. After processing, results can be compared with the database to display driver information.In this process,we used a computer as a server to construct a socket network to realize simultaneous operation in other computers.

During the test, the system had a low recognition rate for the complex Chinese characters 贛Gan(Jiangxi Province), 豫Yu(Henan Province) and 鄂E(Hubei Province).

Since this project was completed in internal computers of neusoft, there is no specific example of works displayed here. Therefore, only the functions of this system are introduced briefly.
