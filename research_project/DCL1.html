    Iterative algorithm is a mathematically approach that can solve optimization problem by starting from a certain initial value X0, according to an iterative formula,and successively calculating X1, X2,..., so that the sequence {Xt} converges to the exact solution for the problem. As figure below shown,the Newton–Raphson method can solve the minimum value of a one-dimensional function which also called Newton iteration method.
